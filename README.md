## About

I'm a frontend developer and engineering intern, currently in my first year of B.Tech in Artificial Intelligence & Machine Learning. I'm nineteen, based in India, and I started shipping production software before my first semester ended.

My engineering approach: understand the problem before writing a line, write code the next person can read without a guided tour, and measure success by whether the product works for the person using it — not just the person who built it.

Long-term, I'm building toward **Full Stack AI Engineering** — clean interfaces, scalable systems, and applied intelligence. I'm not there yet. I'm building deliberately toward it every day.
