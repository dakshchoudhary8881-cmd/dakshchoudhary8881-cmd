## `whoami`

I'm a second-year B.Tech student in AI & ML who decided early on that *learning in isolation doesn't work* — so I build instead. Every concept I study gets turned into a shipped project.

Right now I'm running **two parallel internships**: frontend engineering at Prodesk IT (sprint-based web development) and Python development at 3 Skill (language fundamentals and OOP). I treat both like real engineering jobs — deadlines, documentation, deployment.

The endgame is Full Stack AI Engineering and eventually a startup. I'm not there yet, but I'm building the foundation one sprint at a time.

> Currently in **Sprint 03** of the Prodesk IT curriculum — async JavaScript, REST APIs, geolocation, and intelligent client-side caching.
